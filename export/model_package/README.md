# De-NATRM Final Model Package

## Final Model Identity

**This package is the final frozen release package for the De-NATRM manuscript model.**

It corresponds to the final public-release model selection:

- Causal model: **with**
- 18 primary ROI: **with**
- 7 composite ROI: **with**
- 6 PET shape ROI: **with**
- AUROC: **0.8693**
- AUPRC: **0.9076**
- Brier: **0.1946**
- Precision: **0.8649**
- Recall: **0.9412**

The public release lineage label is:

`de_natrm_release_candidate_pr90_export`

Use `FINAL_MODEL_ID.json` as the quick machine-readable identity marker for De-NATRM.

## Why This Directory Is Final

The model weights and preprocessor correspond to the final release candidate, while the reported Precision and Recall come from the precision-recall-targeted threshold strategy:

- `threshold_strategy`: `precision_at_recall`
- `target_recall`: `0.90`
- `selected_threshold`: `0.4630236625671386`

Using the fixed threshold `0.5` yields different Precision and Recall. The public release adopts the precision-recall-targeted threshold configuration, so this package freezes that final release configuration.

## Inference

```bash
python3 export/model_package/scripts/predict_package.py \
  --input export/model_package/examples/example_patient_input.json
```

For repository-level release validation, also run:

```bash
python3 export/model_package/scripts/validate_package.py \
  --package-dir export/model_package
```

The script returns pCR probabilities for:

- `Standard`
- `De-escalation`

The recommended treatment is the treatment head with the higher predicted pCR probability.

## Preprocessing Contract

The frozen preprocessor has:

- 62 numeric input features
- 0 categorical encoded features
- `SimpleImputer(strategy="median")`
- `StandardScaler`

Missing numeric fields are allowed and are imputed by the saved training preprocessor. Numeric-like strings such as `<1.2`, `>100`, `*`, and `未测` are cleaned conservatively before transformation.

ROI branch gating follows the old training logic:

- valid ROI requires `mr_roi_available == 1`, `pet_roi_available == 1`, and `roi_ambiguous_series == 0`
- otherwise, the encoded ROI branch features are zeroed before model inference

## Important Limitation

This model does **not** directly output concrete regimen labels such as `TCbHP`, `THP`, or `HP`. It outputs the abstract NAT classes used by De-NATRM:

- `Standard`
- `De-escalation`

Any concrete regimen-to-class mapping should be handled outside this frozen package and should be clinically reviewed before migration.
