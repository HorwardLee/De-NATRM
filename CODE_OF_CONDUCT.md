# Code of Conduct

## Our Commitment

We want De-NATRM to be a respectful, useful, and welcoming research repository for collaborators, reviewers, and readers.

## Expected Behavior

Participants in this repository are expected to:

- communicate respectfully
- focus criticism on ideas, evidence, and implementation details
- assume good intent while asking for clarification when needed
- help keep discussions constructive and reproducible
- respect clinical privacy, data boundaries, and release constraints

## Unacceptable Behavior

The following behavior is not acceptable:

- harassment, intimidation, or personal attacks
- discriminatory or demeaning language
- publishing protected data or sensitive patient-related material
- doxxing, threats, or repeated hostile behavior
- intentionally misrepresenting experimental results or repository contents

## Scope

This code of conduct applies to issues, pull requests, review discussions, and other repository-linked interactions.

## Reporting

If you encounter inappropriate behavior, open a private communication channel with the maintainers if possible, or file a minimal public issue that does not escalate harm or disclose sensitive details.

## Enforcement

Repository maintainers may remove comments, close threads, reject contributions, or limit participation when behavior conflicts with this code of conduct or puts the repository at privacy or safety risk.
